# EventF

Bu proje Angular CLI 21.1.3 ile oluşturulmuştur.

## Geliştirme Sunucusu

Yerel geliştirme sunucusunu başlatmak için:

```bash
ng serve
```

Sunucu çalıştıktan sonra tarayıcıda `http://localhost:4200/` adresini aç.

Kaynak dosyalar değiştiğinde uygulama otomatik olarak yeniden yüklenir.

## Kod Oluşturma

Angular CLI component, directive ve pipe gibi yapılar üretmek için komutlar sağlar.

Yeni bir component oluşturmak için:

```bash
ng generate component component-name
```

Kullanılabilir tüm şemaları görmek için:

```bash
ng generate --help
```

## Build Alma

Projeyi build etmek için:

```bash
ng build
```

Build çıktıları `dist/` klasörüne yazılır.

Production build varsayılan olarak performans ve hız için optimize edilir.

## Unit Test Çalıştırma

Vitest test runner ile unit testleri çalıştırmak için:

```bash
ng test
```

## Uçtan Uca Testler

Uçtan uca testler için:

```bash
ng e2e
```

Angular CLI varsayılan olarak uçtan uca test framework’ü ile gelmez.

Projeye uygun bir test aracı ayrıca seçilebilir.
